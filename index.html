Image Tagging Task

Category: Data Collection Time estimate: 5 minutes
Reward: $0.15

You will be shown a series of images. For each image, you need to add appropriate tags that describe the content of the image. Tags should be relevant, specific, and helpful for search and categorization purposes.

Instructions

  1. Examine each image carefully
  2. Add at least 3 relevant tags that describe the image content
  3. Tags should be separated by commas
  4. Use specific and descriptive tags (e.g., "red car" instead of just "car")
  5. Do not include offensive or inappropriate tags
